1. Unzip this folder
2. Install as Chrome extension (see 'textrous-install.png')
	2.1. Open chrome://extensions
	2.2. Enable developer mode
	2.3. Install unzipped package (select this folder)
3. Get Textrous data (https://textrous.irp.nia.nih.gov/)
	3.1. Enter search
	3.2. Click Submit
	3.3. Wait for page to load and for JTLA bar to appear
	3.4. Enter filename (without extension) for this word set
	3.5. Click 'Extract'
	3.6. Click 'Download'
4. Prettify Textrous data
	4.1. Open 'prettify.xlsm'
	4.2. Click 'Enable Content' popup and allow Security exceptions
	4.3. Click 'Prettify' button
	4.4. Selected downloaded Textrous file (<filename>.csv)
	4.5. Wait for your Excel file
